# Frontend-Experiment4: Interactive Image Gallery (JavaScript DOM Manipulation)

## Objectiv
To build an interactive image gallery using HTML, CSS, and JavaScript, where clicking on a thumbnail displays a larger version of the image.

## Files Included
- `index.html` – Structure of the gallery
- `style.css` – Styling for the gallery layout
- `script.js` – JavaScript for thumbnail click functionality

## Experiment Description
This experiment demonstrates:
- Using HTML to create a main image display area and multiple thumbnails.
- Handling user interaction with JavaScript using event listeners.
- Dynamically updating the src attribute of the main image.
- Applying CSS for layout, spacing, hover effects, and basic responsiveness.

## How to Run the project
1. Download or clone the repository:
git clone <https://github.com/BolodevGoyary/Frontend-Experiment4>
2. Ensure the project folder contains:
- index.html
- style.css
- script.js
3. Open index.html in any web browser (Chrome, Firefox, Edge, etc.).
4. Click on any thumbnail to display it as the main image.

## Image Requirements
- Use any images of your choice. Recommended:
- Your own photos
- Free images from Unsplash, Pexels, or Pixabay
- 4 images named: img1.jpg, img2.jpg, img3.jpg, img4.jpg
- Place them inside the images/ folder.
